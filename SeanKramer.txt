 1. What is Semantic HTML? 
 The use of tags which indicate the meaning of the structure (e.g, headers, sections, navs, footer).
    2. What is HTML used for? 
    Structuring the basic content of the page, apart from any styling.
    3. What is an attribute and where do we put it? 
    Attributes provide additional information about HTML elements, and are put within the tags.
    4. What is the h1 tag used for? How many times should I use it on a page?
    Use for the main heading, and only once.
    5. Name two tags that have required attributes
    Anchor <a></a> and image <img>.
    6. What do we put in the head of our HTML document? 
    Metadata, such as <title>, <style>, <meta>, <link>, <script>.
    7. What is an id? 
    An id is a CSS selector, used only once on a page, and the most specific.
    8. What elements can I add an id to? 
    Any elements that can be selected.
    9. How many times can I use the same id on a page? 
    once.
    10. What is a class? 
    A class is a CSS selector that selects elements with a specific class attribute.
    11. What elements can I add a class to? 
    Any elements.
    12. How many times can I use the same class on a page? 
    AS many times as is useful.

    A section is more inclusive; divs mark sub-sections of sections.
    17. What is CSS used for? 
    Styling a web page: e.g, color, fonts, borders.
    18. How to we select an element? Example - every h2 on the page
    Simply by the element itself, e.g., h2.
    19. What is the difference between a class and an id? - Give me an example of when I might use each one
    A class can select multiple elements; and id only one. FOr example, if I want a number of paragraphs to have a certain font, but not all paragraphs, I can select those wanted by giving just those a class. If I have many h2s, and I want just one to have its own color, I can give that one an id.
